## 1. ЦЕЛЬ ПРОЕКТА И ЦЕЛЕВАЯ АУДИТОРИЯ

### Цель сайта

Лендинг — не каталог аренды. Его задача: вызвать желание приехать и сделать один шаг — написать хозяину в Telegram или перейти на Авито.

**Единственная метрика успеха:** клик по CTA-кнопке (Telegram / Авито).

Конкретнее:
- Передать атмосферу места за первые 5 секунд
- Закрыть возражение "далеко" (логистика)
- Убрать неопределённость (цена, что включено, как бронировать)
- Передать доверие к хозяину (не анонимный сервис)

### Целевая аудитория

**Первичная:** Пары, 27–40 лет, Москва. Хотят выбраться на выходные, устали от города. Бюджет 15–25 000 ₽ на двоих за выходные — комфортный.

**Вторичная:** Небольшие компании 3–4 человека, 28–42 года. Повод: день рождения, просто встретиться, баня как цель поездки.

**Общее:** Ищут не "дачу", а место с атмосферой. Видели Instagram-картинки, хотят "такое же". Принимают решение быстро — в пятницу вечером, по телефону.

**Как принимают решение:**
1. Случайно видят фото в Telegram/Instagram или ищут на Авито
2. Переходят на лендинг или Авито
3. Смотрят фото (5–10 секунд)
4. Проверяют расстояние и цену (следующие 15 секунд)
5. Листают вниз — галерея, баня, что рядом
6. Пишут в Telegram или сразу на Авито

**Устройства:** 75–80% трафика с телефона (метро, перерыв, вечер дома). Мобильная версия — основная.

## 2. ОФФЕР И ПОЗИЦИОНИРОВАНИЕ

### Главный оффер (H1 лендинга)

> **"Два дня в берёзовой роще"**

Подзаголовок:
> 63 км от Москвы · A-frame домик 2024 года · Баня · Костёр · Канал рядом

### Что продаём на самом деле

Не домик в аренду. Продаём **состояние**: выдохнуть, замедлиться, побыть вдвоём или небольшой компанией вдали от городского ритма.

Ключевые образы: берёзы · утренний туман · треск костра · баня вечером · кофе на террасе · гамак · тишина

### Уникальные дифференциаторы объекта

1. **A-frame форма** — редкая, фотогеничная, запоминается, хочется снять для себя
2. **Новый 2024 год** — чисто, никакой "истории арендаторов", современно
3. **Берёзовая роща прямо на участке** — не "в окрестностях", а здесь, из окна
4. **Отдельная огневая зона** с грavel, деревянными скамьями, string lights
5. **Канал им. Москвы — 300 м** — конкретно, не абстрактно
6. **63 км** — "выехал после работы, в 20:00 уже на месте"
7. **Один домик = 10 соток только для вас** — никаких соседей-гостей

## 3. ДИЗАЙН-СИСТЕМА

### Настроение / Mood

**Clean + Warm + Natural.** Интерфейс должен исчезать. Технологии не перетягивают внимание. Ощущение: тёплый журнал про загородную жизнь, не booking-сервис.

Референсы: Kip Hideaways, Firelight Camps, Kinfolk Magazine.  
Избегать: Avito-стиль, e-commerce, "айтишный" дизайн, агрессивные CTA.

### Цветовая палитра

```
Основные:
--color-bg:        #F5EFE0   /* тёплый пергамент — фон большинства секций */
--color-bg-white:  #FDFAF5   /* почти-белый — для чередования секций */
--color-bg-dark:   #1A2416   /* тёмный лесной — hero overlay, footer */
--color-text:      #2C2416   /* тёплый почти-чёрный — основной текст */
--color-text-mid:  #6B5B45   /* средний — подзаголовки, подписи */

Акцент:
--color-accent:    #C8742A   /* янтарный — кнопки, иконки, выделения */
--color-accent-hv: #A85E1F   /* hover-состояние акцента */

Природные:
--color-green:     #2D4A2D   /* тёмно-зелёный — иконки природы, детали */
--color-bark:      #8B7355   /* кора — разделители, вторичные элементы */
--color-sand:      #D4C4A0   /* песок — подложки карточек */
```

### Типографика

```
Заголовки H1–H2:   Playfair Display, Regular/Italic
                   Тёплый, editorial, не корпоративный
                   Размеры: H1 = 56–72px desktop / 36–44px mobile

Подзаголовки H3:   Playfair Display, Regular
                   Размеры: 28–36px desktop / 22–26px mobile

Тело и UI:         DM Sans, Regular/Medium
                   Читаемый, нейтральный — "исчезает"
                   Размеры: body = 16–18px / small = 14px

Pull-цитаты:       Playfair Display, Italic, 24–32px
                   Для атмосферных текстов-сценариев

Буквенный интервал заголовков: letter-spacing: -0.02em
Межстрочный:       line-height: 1.6–1.7 (воздух)
```

**Подключение шрифтов:**
```html
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:ital,wght@0,400;0,500;1,400&family=DM+Sans:wght@300;400;500&display=swap" rel="stylesheet">
```

### Компоненты

**Кнопки:**
```css
/* Первичная CTA */
.btn-primary {
  background: #C8742A;
  color: #FDFAF5;
  border-radius: 6px;
  padding: 14px 32px;
  font-family: 'DM Sans', sans-serif;
  font-weight: 500;
  font-size: 16px;
  letter-spacing: 0.01em;
  border: none;
  transition: background 0.2s ease;
  /* НЕТ box-shadow — утяжеляет натуральный стиль */
}
.btn-primary:hover { background: #A85E1F; }

/* Вторичная (аутлайн) */
.btn-secondary {
  background: transparent;
  color: #C8742A;
  border: 1.5px solid #C8742A;
  border-radius: 6px;
  padding: 13px 30px;
}
```

**Иконки:** Простые линейные SVG в цвете `--color-green` или `--color-bark`. Без emoji в основном интерфейсе.

**Grain texture overlay на hero:**
```css
.hero::after {
  content: '';
  position: absolute;
  inset: 0;
  background-image: url("data:image/svg+xml,..."); /* SVG noise */
  opacity: 0.04;
  pointer-events: none;
  mix-blend-mode: overlay;
}
```

**Разделители секций:** Органический SVG-срез или плавная кривая — не прямые линии.

**Whitespace:** Вертикальные padding секций — минимум 100px desktop, 60px mobile.

### HEADER (Sticky)

**Назначение:** Навигация и постоянный доступ к главному CTA.

**Содержимое:**
- Логотип / название: «Репка» (wordmark, Playfair Display Italic)
- Навигация: Фото · Удобства · Цены · Отзывы *(только десктоп)*
- CTA-кнопка: **«Написать хозяину»** → Telegram (всегда видна)

**Поведение:**
- На старте: прозрачный, поверх hero (текст белый)
- После скролла 100px: фон `#F5EFE0`, тень, текст `#2C2416`
- Высота: 64px desktop / 56px mobile
- Mobile: гамбургер → drawer с якорями и кнопкой Telegram

**Фото:** нет

### БЛОК 1 — HERO (Первый экран)

**Цель:** Захватить внимание за 3 секунды. Ответить: "Хочу ли я здесь остаться?"

**Контент:**

```
H1:          Два дня в берёзовой роще
Subhead:     63 км от Москвы · A-frame домик 2024 года · Баня · Костёр · Канал рядом
CTA 1:       Написать хозяину  →  [Telegram]
CTA 2:       Посмотреть фото   →  [якорь на галерею]
```

**Дизайн:**
- Полноэкранный (100vh), фото или видео-петля фоном
- Тёмный overlay градиент снизу: `rgba(26,36,22, 0.35)` → `rgba(26,36,22, 0.7)`
- Весь текст и кнопки — белый / `#FDFAF5`
- H1 центр или левый блок — не мелкий, 64px desktop / 40px mobile
- Grain texture overlay поверх фото (opacity 0.04)
- Кнопки рядом горизонтально (desktop) / вертикально (mobile)

**Анимация:** H1 fade-in снизу (0.8s, ease-out) после загрузки страницы. Подзаголовок — с задержкой 0.2s. Кнопки — 0.4s.

**Фото / видео для hero:**

| Вариант | Описание | Приоритет |
|---|---|---|
| **Основное фото** | A-frame фасад в золотой час (−30 мин до заката). Берёзы в кадре, забор вне кадра. Тёплый свет. | P0 — сделать до запуска |
| **Видео-петля (опция)** | 15–20 сек без звука: ветер в берёзах → домик → дым из трубы бани → закат. Auto muted loop. | P1 — если будет видео |
| **Зимний вариант (сезонный)** | Домик в снегу, тёплый свет в окнах, вечер | P2 — для зимы |

> ⚠️ Текущие фотографии: видны соседские заборы и строения. Для hero нужна перефотосъёмка с правильными ракурсами (см. раздел 8).

### БЛОК 2 — АТМОСФЕРА / СЦЕНАРИЙ

**Цель:** Продать эмоцию, не квадратные метры. Клиент должен мысленно оказаться там.

**Контент:**

```
H2:      Приехать в пятницу. Выдохнуть в субботу.

Текст (pull-quote):
  «Вечером — баня и костёр в берёзах.
   Утром — тишина, кофе и вода в 300 метрах.
   Никаких планов. Только выходные.»

Теги (4 штуки):
  ○ Для пары       ○ Компания до 4
  ○ Тихий отдых    ○ Активный уикенд
```

**Дизайн:**
- Светлый фон `#FDFAF5`, много воздуха
- Pull-quote Playfair Display Italic, 26–28px, цвет `--color-text-mid`
- Теги — простые outlined capsule: border `--color-bark`, текст `--color-text`, фон прозрачный
- Справа или снизу (десктоп / мобайл) — одно атмосферное фото

**Фото для блока:**

| Описание | Характеристика |
|---|---|
| Вид из треугольного окна A-frame на берёзы | Интерьер + природа = одновременно |
| ИЛИ: пара с кофе на террасе утром | Lifestyle — идентификация ЦА |
| ИЛИ: гамак между берёзами с листьями | Спокойствие, воздух |

**Цель:** Визуальное доверие. Снять страх "на фото лучше, чем в реальности".

**Контент:** 10–16 фотографий. Категории и порядок:

| № | Категория | Что должно быть на фото | Приоритет |
|---|---|---|---|
| 1 | **Фасад — главное фото** | A-frame + берёзы, золотой час | P0 |
| 2 | **Фасад — альтернатив** | Другой ракурс / другой сезон | P1 |
| 3 | **Терраса + deck** | Egg chair, deck, вид на рощу | P0 |

| 6 | **Вид из окна** | Треугольное окно A-frame → берёзы | P0 |
| 7 | **Огневая зона** | Костёр горит, люди сидят, string lights, сумерки | P0 |
| 8 | **Баня снаружи** | Строение + вечерний свет | P1 |
| 9 | **Баня внутри** | Парная, веники, тёплый свет | P1 |

| 12 | **Мангальная зона** | Мангал с казаном, компания | P2 |

**Дизайн галереи:**
- Masonry grid или 3-column grid с одним большим главным фото слева
- Кнопка "Все фото" → lightbox с fullscreen просмотром

- Формат WebP, максимум 200 КБ на превью, 800 КБ fullscreen

**Анимация:** Staggered entrance — карточки появляются последовательно при скролле (задержка 0.08s между карточками).

## 5. АНИМАЦИИ И ИНТЕРАКТИВНОСТЬ

### Принципы

- Анимации усиливают настроение "замедления" — медленные, плавные
- Используется `prefers-reduced-motion: reduce` — все анимации отключаются
- Без агрессивного параллакса (мешает мобайлу, раздражает)
- Без magnetic cursor, без WebGL-эффектов — перегружают атмосферу

### Scroll-triggered fade-in (все секции)

```css
.reveal {
  opacity: 0;
  transform: translateY(24px);
  transition: opacity 0.7s ease-out, transform 0.7s ease-out;
}
.reveal.visible {
  opacity: 1;
  transform: translateY(0);
}

@media (prefers-reduced-motion: reduce) {
  .reveal { opacity: 1; transform: none; transition: none; }
}
```

```js
const observer = new IntersectionObserver(
  entries => entries.forEach(e => e.isIntersecting && e.target.classList.add('visible')),
  { threshold: 0.15 }
);
document.querySelectorAll('.reveal').forEach(el => observer.observe(el));
```

### Staggered entrance (карточки удобств, галерея)

```css
.card:nth-child(1) { transition-delay: 0.00s; }
.card:nth-child(2) { transition-delay: 0.08s; }
.card:nth-child(3) { transition-delay: 0.16s; }
/* и т.д. */
```

### Hero видео-петля (если будет видео)

```html
<video autoplay muted loop playsinline poster="hero-poster.webp">
  <source src="hero-loop.mp4" type="video/mp4">
</video>
```
- Без звука (muted обязательно)
- Poster = статичное фото (показывается до загрузки видео)
- Размер видео: сжать до < 5 МБ (720p достаточно)

### Sticky header transition

```css
.header {
  transition: background 0.3s ease, box-shadow 0.3s ease;

.header.scrolled {
  background: #F5EFE0;
  box-shadow: 0 1px 12px rgba(44,36,22,0.08);
}
```

## 7. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### Performance

- **LCP** (Largest Contentful Paint): < 2.5 сек
- Hero фото: WebP, максимум 300 КБ (сжать через squoosh.app или sharp)
- Все фото галереи: WebP, lazy loading (`loading="lazy"`)
- Шрифты: preload через `<link rel="preload">`, display=swap
- Видео hero: < 5 МБ, mp4 h.264

<title>A-frame домик Репка — аренда посуточно, Дмитровский район</title>

### Мобильный дизайн (Mobile-First)

- Breakpoints: 375px / 768px / 1280px
- Все кнопки: минимум 44×44px (thumb-friendly)
- Шрифт body: минимум 16px (без zoom на iOS)
- Hero H1 на мобайле: 36–40px
- Галерея: горизонтальный swipe на мобайле
- Карта: уменьшается до 250px высоты
- Sticky header: компактный, только лого + кнопка Telegram

### Стек

Сайт пишется вручную (Claude генерирует код). Стек — минимальный:

- **HTML + CSS + Vanilla JS** — без фреймворков, без сборщиков
- Один файл `index.html` + `style.css` + `script.js`
- Никаких зависимостей npm, никакого build pipeline
- Хостинг: GitHub Pages / Netlify / Vercel (drag&drop deploy)
